# 概要

このアプリは、チーム内でのタスクの共有と進捗管理を効率化するために開発されました。
ユーザーは複数のルームに参加し、タスクを作成、編集、削除、進行状況を確認できます。

## 主な機能

アプリで提供する主な機能を箇条書きで紹介します。

- ユーザー認証（ログイン/サインアップ）
- タスクのCRUD操作
- ルームごとのタスク管理
- ルームへの参加
- タスクのロック機能
- 管理者によるルーム参加の承認(未実装)
- ステータス、カテゴリのCRUD操作(未実装)
